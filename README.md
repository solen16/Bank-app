# Bank-app

# FR Version : 
Ce projet consiste en la conception d’une base de données bancaire complète, modélisant les clients, comptes, contrats et opérations financières. Réalisé à travers les étapes MCD, MLD et MPD, il implémente des contraintes d’intégrité et des relations complexes. Le système permet de gérer les transactions (paiements, virements), les produits d’épargne et les emprunts via SQL sous Oracle. 

# EN Version :
This project focuses on designing a complete banking database modeling clients, accounts, contracts, and financial operations. Built through conceptual, logical, and physical data models, it enforces data integrity and complex relationships. The system manages transactions (payments, transfers), savings products, and loans using SQL with Oracle.

Trois scripts permettent la création et la lecture des données de cette application afin d'en simuler l'utilisation et un script permet la désinstaller: 
- Script de la création de la base de données
- Script de saisie des données dans la base de données
- Script de lecture des informations sur les clients
- Script de désinstallation de la base de données de manière propre
